## Steganography

### Steganography-Tools

1. [stegsolve.jar](http://www.caesum.com/handbook/stego.htm) </br>

Run this command to install stegsolve.
```
wget http://www.caesum.com/handbook/Stegsolve.jar -O stegsolve.jar
chmod +x stegsolve.jar
mkdir bin
mv stegsolve.jar bin/
```
